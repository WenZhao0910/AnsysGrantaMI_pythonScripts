Streamlined API Example Notebooks

The MI Scripting Toolkit for Python provides you with ready-to-use sample code to help you get started, whether you’re a new user or upgrading from the Foundation API. All examples use the MI Training database, available from the Ansys Download Center or the Download Software page on the Granta MI Support site .
