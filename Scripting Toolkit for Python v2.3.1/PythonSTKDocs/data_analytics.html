Data Analytics Examples¶

These examples show how to use the MI Scripting Toolkit to perform some common data analytics tasks, including statistical roll-ups and generation of plots. All examples use the MI Training database, available from the Ansys Download Center or the Download Software page on the Granta MI Support site Most examples also require the use of additional dependencies outside of the STK, including pandas, numpy, matplotlib, and seaborn. Download all the notebooks and the example files here.
