Streamlined API Example Notebooks
=================================


The MI Scripting Toolkit 2.3 for Python provides you with ready-to-use sample code to help you get started, whether you're a new user or upgrading from the Foundation API. All examples use the MI Training database, available from the Ansys Download Center or the Download Software page on `the Granta MI Support site <https://grantadesign.com/industry/support/granta-mi>`_. Download all the notebooks and the example files :download:`here <samples/streamlined/example_files.zip>`.
